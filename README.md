# Product Management Dashboard

**Sinh viên:** NguyenHoangMinhTri-2280603364

## Mô tả dự án
Dashboard quản lý sản phẩm sử dụng Platzi Fake Store API với đầy đủ chức năng CRUD, tìm kiếm, sắp xếp, phân trang và export CSV.

## Công nghệ sử dụng
- HTML5
- CSS3
- Vanilla JavaScript
- Bootstrap 5
- Platzi Fake Store API

## Chức năng
1. ✅ Hiển thị danh sách sản phẩm từ API
2. ✅ Tìm kiếm theo tên sản phẩm (real-time)
3. ✅ Sắp xếp theo Title và Price
4. ✅ Phân trang (5, 10, 20 items/page)
5. ✅ Export CSV
6. ✅ Xem chi tiết sản phẩm
7. ✅ Chỉnh sửa sản phẩm
8. ✅ Tạo sản phẩm mới
9. ✅ Hiển thị mô tả khi hover (Bootstrap Tooltip)
10. ✅ Loading states và error handling

## Cách chạy
Mở file `index.html` trực tiếp trong trình duyệt web.
